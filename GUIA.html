⚙️ Cómo Funciona el Sistema de Rutas

Concepto clave: Un único archivo index.php maneja todas las rutas del proyecto. 
1. Usuario abre: http://localhost/academia/?page=catalogo

2. Apache recibe la solicitud
   ↓
3. .htaccess redirige a index.php
   ↓
4. index.php lee: $_GET['page'] = 'catalogo'
   ↓
5. Busca en el mapeo de páginas:
   $pages['catalogo'] = 'catalogo/PROJECT/catalogo.php'
   ↓
6. Incluye el archivo y muestra el contenido
   ↓
7. Usuario ve: Catálogo funcional con BD

⚡ Configuración de Apache (.htaccess)

<IfModule mod_rewrite.c>
    RewriteEngine On
    RewriteBase /academia/
    
    # Permitir archivos y carpetas reales
    RewriteCond %{REQUEST_FILENAME} !-f
    RewriteCond %{REQUEST_FILENAME} !-d
    
    # Todo lo demás → index.php
    RewriteRule ^(.*)$ index.php?page=$1 [QSA,L]
</IfModule>

Esto permite que URLs limpias se traduzcan a parámetros GET.

🔧 Solución de Problemas

❌ Error: "Not Found"

Causa: Apache no encuentra el archivo o la reescritura de URLs falla
Solución:
1. Verifica que `.htaccess` esté en la raíz de `academia/`
2. Asegúrate que `mod_rewrite` está habilitado en Apache
3. Comprueba que `index.php` existe y es legible

❌ Error: "Conexión a BD fallida"

Causa: MySQL no inició o credenciales incorrectas
Solución:
1. Inicia MySQL en XAMPP
2. Verifica credenciales en `backend/conexion.php`
3. Asegúrate que la BD `proyectotam` existe

❌ CSS/JS no se cargan

Causa: Rutas relativas incorrectas
Solución:
1. Usa rutas relativas desde la raíz: `css/style.css`
2. O usa rutas absolutas con BASE_URL
3. Verifica que los archivos existan en las carpetas correctas

✅ Lo que se Corrigió

Problema 1: Múltiples archivos `index.php` e `index.html`
Solución: Un único `index.php` como punto de entrada centralizado
Problema 2: Rutas rotas en HTML y PHP
Solución: Uso de rutas relativas consistentes y BASE_URL
Problema 3: Apache no sabía cómo enrutar solicitudes
Solución: `.htaccess` configurado correctamente con `mod_rewrite`
Problema 4: Error 404 en páginas del catálogo
Solución: `catalogo.php` (versión funcional) incluida y accesible
Problema 5: Estructura confusa y desorganizada
Solución: Estructura clara con carpetas específicas y documentación
